# AI-Food-Checker
Using AI chart box describe the all about that food

1. User Interface (Front Page)
The website shows a search box, dropdown list, and food cards.
Users can:
* Type the name of an Indian dish (example: Samosa).
* Select a dish from the dropdown list.
* Click any dish from the food database grid.

2. Food Selection
When the user selects or searches for a food item:
* The website checks that food in its food database (JavaScript object).
Each food already has:
* Health status (Healthy / Moderate / Unhealthy)
* Ingredients list
* Nutrition values (calories, carbs, fat, protein)
* Health tips

3. Food Search System
As the user types:
The website shows live search suggestions.
It matches:
* Food name
* Alternate names (aliases)
* Ingredients
This helps users find dishes even if they type different names (e.g., Golgappa ‚Üí Pani Puri).

7. Health Tips Section
Each dish shows simple health advice, for example:
* Eat in small quantity
* Use less oil or sugar

This guides users to make healthier food choices.

8. Food Database Section
The bottom section shows Indian dishes as cards with images.
Each card shows:
* Food name
* Health tag (Healthy / Moderate / Unhealthy)
Users can filter dishes using buttons:
* All
* Healthy
* Moderate
* Unhealthy

9. Smart Filtering
When a filter button is clicked:
* Only selected category foods are shown.
* This helps users quickly find healthy food options.
